## 食用指南

### Homework

用于存储平时作业的代码

因为估计总的作业量不会很多，所以Homework目录下不再细分

命名规则：num1-num2.cpp, num1代表第num1次作业，num2代表第num2题，至于每一题里的小问，就通过函数的命名来区分，第一问p1，第二问p2，依此类推......
